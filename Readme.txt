JavaScript Log

1) You can easy show/hide log in console area.
2) Easy to maintain.
3) We have colour mapping for log/warn/error.
4) Easy to switch the mode like Production or Development.
5) You can set prefix and postfix message before/after the console in log/warn/error.


Production:
 Error log is Disable.
 
Development:
 Error log is Enable.

Requirements: 
 Install Jquery (https://jquery.com/download/)
