Implemented Javascript Log

How to open the Javascript console:

  • Chrome/Windows: CTRL+SHIFT+J
  • Chrome/Mac: CMD+ALT+J
  • Firefox/Windows: CTRL+SHIFT+I
  • Firefox/Mac: CMD+ALT+K
  • Install Firebug on Firefox
  • Internet Explorer: F12 to open Developer Tools, then click the Console tab or button

Easy to use:

  • Easy to show/hide log in console area.
  • Easy to maintain.
  • Easy to switch the mode like Production(pro) or Development(dev).
  • Easy colour map for dev and pro for log/warn/error. Color map is only available when log is enable.
  • Easy to set prefix and postfix message in log/warn/error message.
  • Easy to set a global variable to check in console area(F12). Global variable functionality only work. When log is enabled. All global variable start with nn_js_global_.
  • Easy to show alert. Alert only available when log is enable.

Requirements:

  • Install Jquery (https://jquery.com/download/)
  • Install Javascript Log manager js of specific version.

Feature

Pro:
Error log is Disable.
Dev:
Error log is Enable.

Create a variable within the head tag

var logObj=NN_JS_Log(); //default mode: dev and color: true

Switch the mode from dev to pro.

var logObj=NN_JS_Log({mode:'pro'});

V 1.0.0

Property
Property Name Description
mode Tell which mode should be loaded.
color Color log/warn/error should be enable or not.
show_log Log should be enable or not.
prefix Set message for prefix. What to show before console. (default: null)
repeat How many time to repeat the message of prefix and suffix. (default: 1)
suffix Set message for suffix. What to show after console. (default: null)
Method
Method Name Parameter Description
log mgs Print mgs on console.
warn mgs Print mgs on console.
error mgs Print mgs on console.
global variableName,variableValue Create global variable, when log is enable. Global name start with nn_js_global_{variableName}.
alert mgs Show the alert.
globalDef void Show all the global variable in console(Array).

snippet

var logObj=NN_JS_Log({color:true});
logObj.prefix='---- prefix ----';
logObj.suffix='---- suffix ----';
logObj.repeat=3;
logObj.log('Log');
logObj.prefix=null;
logObj.warn('Warn');
logObj.error('Error');
var a='global testing';
logObj.global('test',a);
logObj.global('test1','global_testing');
logObj.alert('Js manager test');
logObj.globalDef();
